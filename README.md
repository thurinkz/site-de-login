# 🔐 Sistema de Login + Dashboard

Projeto de autenticação com dashboard, desenvolvido para simular um fluxo real de login em aplicações web.

## 🚀 Acesse o projeto
👉 https://thurinkz.github.io/site-de-login/

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript

## ⚙️ Funcionalidades
- Validação de e-mail com regex
- Validação de senha (mínimo de caracteres)
- Mostrar/ocultar senha
- Feedback visual de erro e sucesso
- Simulação de autenticação de usuário
- Persistência de sessão com sessionStorage
- "Lembrar e-mail" com localStorage
- Redirecionamento após login
- Dashboard protegido (acesso apenas logado)
- Exibição de dados do usuário no dashboard
- Logout funcional

## 💡 O que este projeto demonstra
- Manipulação do DOM com JavaScript
- Controle de estado da interface (loading, erro, sucesso)
- Uso de armazenamento no navegador (localStorage e sessionStorage)
- Estruturação de páginas e organização de código
- Criação de interfaces interativas

## 🎯 Objetivo
Desenvolver habilidades em front-end criando projetos que simulam funcionalidades reais de sistemas web.

## 📌 Observação
Este projeto simula um sistema de login apenas no front-end, sem integração com backend.
